## Trader Behavior vs Market Sentiment (Web3 Trading)

This project analyzes how trader profitability, risk-taking,
and behavior change across Fear and Greed market regimes
using Hyperliquid trading data and the Bitcoin Fear & Greed Index.

### Key Insights
- Fear regimes show higher risk-adjusted returns
- Greed regimes exhibit overtrading and higher volatility
- Sentiment is a statistically significant driver of outcomes

### Structure
- notebook_1.ipynb: Full analysis (Google Colab)
- csv_files/: Processed datasets
- outputs/: Report-ready visualizations
- ds_report.pdf: Executive summary & recommendations

### Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, SciPy
